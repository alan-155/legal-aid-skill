# legal-aid-skill

## What This Skill Does

Perform initial legal aid intake when someone needs legal help but can't afford a lawyer. Use this skill whenever a person describes a legal problem, asks about their rights, mentions court dates, eviction, debt collection, custody, immigration status, benefits denial, or says they "need a lawyer" but can't afford one. Always use this skill before giving any legal information — it sets the right tone, gathers the right facts, and routes the person appropriately.  It is aimed to help people in need of legal aid that may not be able to afford it and aims to direct them to the right resources.  


## How to Use

1. Download the `SKILL.md` file from this repository
2. Go to claude.ai → Settings → Skills
3. Upload the `SKILL.md` file
4. Toggle the skill on
5. Start a new conversation and try one of the sample prompts below

## Sample Prompts

Try these prompts to test the skill:

- "[I can't afford a lawyer]"
- "[I got served an eviction notice, what are my rights?]"
